# Pawosa Shopify Theme

This repository contains the source code for the Shopify theme used on Pawosa (`pawosa.com`).

## Directory Structure
- `assets/`: Theme stylesheets, JavaScript, and image assets.
- `config/`: Theme settings and configuration files.
- `layout/`: Main layout files (e.g., `theme.liquid`).
- `locales/`: Localization files for different languages.
- `sections/`: Section templates that can be customized in the Shopify admin.
- `snippets/`: Reusable Liquid helper code snippets.
- `templates/`: Page and collection template JSON files.
